Пользователи должны иметь возможность:

- Просмотр оптимального макета для каждой страницы в зависимости от размера экрана их устройства
- Просматривайте состояния наведения для всех интерактивных элементов на странице.
- Увидеть собственный IP-адрес на карте при начальной загрузке страницы
- Поиск любых IP-адресов или доменов и просмотр ключевой информации и местоположения

Ваша задача состоит в том, чтобы создать это приложение для отслеживания IP-адресов и 
максимально приблизить его к дизайну. Чтобы получить местоположения IP-адресов, вы будете 
использовать [IP Geolocation API от IPify] (https://geo.ipify.org/). Для создания карты мы 
рекомендуем использовать [LeafletJS](https://leafletjs.com/).

⚠️ **ВАЖНО** ⚠️: Чтобы использовать API IP-геолокации от IPify, вам необходимо зарегистрировать 
бесплатную учетную запись. Для этого вам не нужно добавлять какие-либо данные о картах, и это очень
 быстрый процесс. Это сгенерирует для вас ключ API. Обычно вы можете ограничить свой API-ключ 
определенным URL-адресом (вашим собственным доменом). Это гарантирует, что другие люди не смогут 
использовать ваш ключ API на своих сайтах. В IPify нет этой функции, но, поскольку вы не добавляете 
данные своей карты, это не проблема. **Поэтому обязательно зарегистрируйте только бесплатную учетную 
запись и НЕ вводите данные карты**.

В качестве API сопоставления мы рекомендуем использовать [LeafletJS](https://leafletjs.com/). 
Это бесплатно для использования и не требует ключа API. Если вы решите использовать другой API, 
например Google Maps или Mapbox, обязательно защитите свой ключ API. Вот руководства для Google Maps 
и Mapbox, обязательно внимательно прочитайте их:

– [Передовые практики для ключей API от Google Developers](https://developers.google.com/maps/api-key-best-practices)
- [Как безопасно использовать Mapbox](https://docs.mapbox.com/help/troubleshooting/how-to-use-mapbox-securely/)

Публичное раскрытие вашего ключа API может привести к тому, что другие люди будут использовать 
его для отправки запросов для своего собственного приложения, если не будут приняты надлежащие 
меры предосторожности. Пожалуйста, внимательно прочитайте руководства и следуйте их рекомендациям.
